기본 견적 작성 순서
1
차종 선택돌핀·아토3·씰·씨라이언7 중 원하는 차종을 탭하세요
2
트림 선택차종별 트림(등급)과 가격을 확인하고 선택하세요
3
색상 선택트림별 제공 색상 중 고객이 원하는 색상을 선택하세요 (추가비용 없음)
4
할인 입력 (선택)영업 할인 금액을 입력하면 차량가격에서 차감되어 취등록세·공채·할부원금이 모두 재계산됩니다
5
보조금 입력지역 선택 시 서울·인천·김포는 국고/지방비/전환지원금 자동 입력됩니다. 다자녀·생애첫차 항목도 체크하세요
6
등록·부대비용 확인취등록세는 VAT 제외 차량가 × 7% − 140만원(전기차 감면)으로 자동 계산됩니다
7
구매 방식 선택현금·할부 중 선택. 할부는 금리(0.1% 단위)와 개월 수를 조절하세요
8
견적서 출력·저장우측 패널 하단 버튼 또는 모바일 하단 바에서 출력/저장
출력 및 저장 방법
🖨
인쇄/PDF
출력 버튼 → 새 창 → [인쇄/PDF 저장] 클릭 → 브라우저 인쇄 창에서 "PDF로 저장" 선택
📥
이미지 저장 (PC)
출력 버튼 → [이미지 저장] 클릭 → PNG 파일 자동 다운로드
📱
이미지 저장 (모바일)
출력 버튼 → 견적서 이미지를 길게 눌러서 사진 저장
📱 앱으로 설치하는 방법
1
Safari로 이 페이지 열기반드시 Safari 브라우저를 사용해야 합니다 (Chrome 불가)
2
하단 공유 버튼 탭화면 하단 중앙의 □↑ 아이콘을 탭하세요
3
"홈 화면에 추가" 선택스크롤을 내려 "홈 화면에 추가"를 탭하세요
4
"추가" 탭우측 상단 "추가"를 탭하면 홈 화면에 앱 아이콘이 생깁니다
💡 이제 홈 화면에서 BYD 견적 아이콘을 탭하면 앱처럼 전체 화면으로 실행됩니다!
1
Chrome으로 이 페이지 열기Android Chrome 브라우저를 사용하세요
2
설치 배너가 자동 표시됨화면 하단에 "앱으로 설치하기" 배너가 나타납니다
3
"설치" 버튼 탭또는 Chrome 우측 상단 ⋮ 메뉴 → "앱 설치" 선택
4
홈 화면에서 실행설치 완료 후 홈 화면의 앱 아이콘으로 실행하세요
💡 "설치" 버튼이 보이지 않으면 Chrome 메뉴(⋮) → "홈 화면에 추가"를 선택하세요
1
Chrome 또는 Edge로 열기PC에서 Chrome이나 Edge 브라우저를 사용하세요
2
주소창 우측 설치 아이콘 클릭주소창 맨 오른쪽에 ⊕ 아이콘이 나타납니다
3
"설치" 클릭팝업에서 "설치"를 클릭하면 바탕화면에 앱이 추가됩니다
💡 설치 후 Windows 시작메뉴나 macOS Launchpad에서도 앱으로 실행할 수 있습니다
